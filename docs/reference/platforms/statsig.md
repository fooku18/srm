# Statsig

Default SRM Checks
{: .label .label-green }


[Statsig](https://www.statsig.com) automatically and continuously verifies and warns users about Sample Ratio Mismatch. Documentation is [available here](https://docs.statsig.com/experiments-plus/monitor).
